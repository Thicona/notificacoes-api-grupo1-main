# Pesquisa de Mercado — Serviços de Notificação

## Serviços de E-mail Transacional

| Serviço    | Plano Gratuito           | Preço Inicial | Diferenciais           |
| ---------- | ------------------------ | ------------- | ---------------------- |
| SendGrid   | 100 emails/dia           | US$ 15/mês    | API robusta, templates |
| Mailgun    | 5.000/mês (3 meses)      | US$ 35/mês    | Foco em devs           |
| Amazon SES | 62.000/mês (se usar EC2) | US$ 0.10/1000 | Escala, preço          |
| Mailtrap   | 500/mês (teste)          | US$ 15/mês    | Sandbox para testes    |

## Como o nosso projeto se compara?

Nosso projeto é semelhante a serviços como SendGrid e Mailgun
por permitir o envio automatizado de notificações e mensagens. 
Porém, enquanto essas plataformas oferecem infraestrutura completa, 
escalabilidade e métricas avançadas, nosso sistema é mais simples, 
focado na integração com a aplicação e no baixo custo.

## O que poderíamos adotar no futuro?

No futuro, poderíamos adicionar:

-templates personalizados;
-dashboard de métricas;
-histórico de notificações;
-envio por SMS ou WhatsApp;
-automação de mensagens;
-maior segurança e escalabilidade usando serviços como Amazon SES.
