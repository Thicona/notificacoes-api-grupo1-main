# Notificações API

Projeto desenvolvido para fins acadêmicos no SENAI.

## Tecnologias Utilizadas

* Node.js
* Express
* Sequelize
* MySQL

## Instalação

```bash
npm install
```

## Executando o projeto

```bash
npm start
```

## Executando as migrations

```bash
npm run db:migrate
```

## Executando os seeds

```bash
npm run db:seed
```
